Tipografias en html

Código: Como se usa: como se ve:
strong < strong > texto < /strong> Ejemplo
b < b > texto < /b> Ejemplo
I < I > texto < /I> Ejemplo
em < em > texto < /em> Ejemplo
big < big > texto < /big> Ejemplo
small < small > texto < /small> Ejemplo
sub < sub > texto < /sub> H 2 O
sup < sup > texto < /sup> 3 2 X
strike < strike > texto < /strike> Ejemplo
U < u > texto < /u> Ejemplo

Notas:



Notas:

B: Sirve para darle negrilla al texto

I: Sirve para deladear el texto

em: Sirve para deladear el texto

big: Sirve para agrandar el texto

small: Sirve para minimizar el texto

sub: Sirve para poner un subIndice en el texto

sup: Sirve para poner un superIndice en el texto
