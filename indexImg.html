Manejo de Imagenes

Imagenes

Niveles: Código: Imagen:
Nivel 1 < img src="img/example.jpg" >
Nivel 2 < img src="carpeta/carpeta2/example.jpg">
1 Nivel abajo < img src="../example.jpg">
2 Niveles abajo < img src="../../example.jpg">

Notas:



Img: La etiqueta img, se usa para extraer una imagen dentro del pc, por medio de las diferente rutas:


Carpeta/img.jpg: se usa para extraer una imagen dentro de una carpeta.

../img.jpg: Se usa para extraer una imagen fuera de la carpeta de trabajo:

../../img.jpg: Se usa para extraer una imagen fuera de la carpeta de trabajo 2 niveles abajo :


Manejo del tamaño de las imagenes:


Widht: Se usa para ponerle ancho a la imagen
heigth: Se usa para ponerle altura la imagen

Formatos de imagen

Niveles: Soporte: Imagen:
.GIF Soporta transaprencias,
Soporta animaciones simples,
No tiene perdida de calidad
.JPG Utilizadas para fotografias
Soporta millones de colores
aplica compresion
.PNG El más recomendable para imágenes que no sean fotografias
Ideal para dibujos o capturas de pantalla
Soporta transparencia y niveles de transpariencia(png-24)
.SVG Son vectoriales
No tienen pérdida de calidad al aumentar de tamaño
Pesan muy poco.
No Example

Notas:




Para que funciones solo hay que cambiar el (.JPG) en la direccion del Img, por la direccion que se desee (.PNG, .GIF, .SVG )
