Manejo de Videos

Formatos de Video

Formato: Definicion:
Mp4 El formato mp4 (ficheros con extencion .mp4 o .mov) que bajo licencia utiliza el códec h264
(el mismo que utiliza flash). es impulsado por Apple y Microsoft
OGG El formato OGG (ficheros con extensión .ogv) que utiliza el códec On2vp8 es gratuito
impulsado por Mozilla, Opera y Firefox
WebM El formato WebM (fichero con extencion .webm) también es gratuito e impulsado por
los mismos que apoyan el formato OGG (Mozilla, Opera y Firefox)

Notas:



Video y Source:

La etiqueta Video, se usa para expresar que acontinuacion se usara un video,
la etiqueta Source, se usa para extraer el video dentro del equipo



iFrame:

La etiqueta iFrame, se usa para traer un video de youtube, copiando la direccion que youtube mismo te da.

Html Estandares

Formato: Definicion:
Width: Indica la anchura del vídeo, expresada en píxeles (px) o en porcentaje (%)
Height height: Indica la altura del vídeo, expresada en píxeles (px) o en porcentaje (%)
Controls Establece si tendrá la barra de navegación (play, stop, volumen...) o no
Autoplay Indica si el vídeo se inicia detenido o ya aparece por defecto reproduciéndose
Loop Establece si al acabar el vídeo, éste debe volver a empezar o no
Muted Indica si el audio se reproduce o se reproduce sin audio
Preload Indica si el vídeo se empieza a cargar cuando el navegador carga la página html
Poster: Da la posibilidad de establecer una imagen (jpg, gif o png) como fotograma inicial del video

Notas:

Ejemplo del Codigo

< video width="800" height="600" controls autoplay loop poster="fotograma.png"> < source src="video.ogv" type="video/ogg"/> < /video>
